# ByteTrail Privacy

ByteTrail is local-first storage analysis software.

- All scanning, source attribution, rule evaluation, and cleanup validation happen locally on the Mac.
- File paths, file names, installed application lists, scan results, cleanup history, and recovery records are not uploaded.
- No account is required.
- ByteTrail contains no telemetry, analytics, advertising SDK, remote rule service, or cloud feature.
- ByteTrail does not use an AI model, online lookup, remote classification, or network request to decide what is safe to review or clean.
- The app reads file-system metadata and installed-app bundle metadata needed to classify, attribute, and size a finding. It does not inspect document contents for classification.
- Cleanup history is stored locally and contains only the metadata needed to explain an operation and offer recovery.
- User data is not sold, transmitted, or used for advertising.

ByteTrail has no network entitlement, contains no network client integration, and does not require an external server.
